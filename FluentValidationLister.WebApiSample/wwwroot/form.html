Validation demo

This is intended to demonstrate dynamic clientside validation rules that are defined when the page initially loads.

NOTE: This form implements clientside validation for a number of rules (server-side validation is also presented the same way). Open your preferred developer console to see whether the validation is from the client or server (naturally if clientside validation fails, server-side validation will not occur; unless the checkbox is unticked of course).

This demo is implemented using jQuery as a very simplified example - you should extend it for production use, for example to allow it to validate each field during the "onBlur" event.
